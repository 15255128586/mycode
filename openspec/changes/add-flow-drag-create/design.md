## Context
流程图编辑需要更高效的节点创建方式，尤其在长流程中拖线到空白创建节点是更自然的交互。

## Goals / Non-Goals
- Goals: 支持拖线到空白创建节点；创建后自动连线并吸附网格；可取消创建。
- Non-Goals: 不引入完整画布编辑器或复杂节点模板库。

## Decisions
- Decision: 使用 React Flow 的连接结束事件识别空白落点，并弹出轻量创建弹窗。
- Decision: 新节点位置按当前网格吸附规则计算，保持与现有布局一致。

## Risks / Trade-offs
- 误触创建 → 弹窗提供明显取消，且默认不提交即不创建。
- 画布交互复杂度提升 → 保持弹窗最小字段（类型、标题）。

## Migration Plan
- 增加事件处理与弹窗组件；不改变现有节点数据结构。
